# Retrospective Cognition Workflow Asset

Status: source of truth
Language: English
Last updated: 2026-03-26

## Workflow

1. Identify the completed work round.
2. Reload the relevant truth layers.
3. Separate:
   - verified facts
   - inference
   - misread state
4. Decide what should persist.
5. Place each lesson into the correct durable layer.
6. Mark maturity when the lesson is not yet stable.
7. Keep public-safe and internal-only surfaces separate.

## Output Expectation

The workflow should produce a bounded result that includes:

- verified state
- misread state
- durable lesson
- current operating rule
- destination layer or file
