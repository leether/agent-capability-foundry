## Workflow

1. 识别这次已完成的工作轮次。
2. 重载相关 truth layers。
3. 分开：
   - verified facts
   - inference
   - misread state
4. 判断什么应该持久保留。
5. 把每条经验放到正确的 durable layer。
6. 如果经验尚未稳定，标出 maturity。
7. 保持 public-safe surface 和 internal-only surface 分离。

## 输出预期

这个 workflow 应产出一个有边界的结果，至少包括：

- verified state
- misread state
- durable lesson
- current operating rule
- destination layer 或 file
