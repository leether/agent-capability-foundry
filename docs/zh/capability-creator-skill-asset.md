# Capability Creator Skill 资产

状态：派生文档
语言：简体中文
英文真源：[capability-creator-skill-asset.md](../en/capability-creator-skill-asset.md)
最后更新：2026-03-26

## 用途

当一个新 capability 需要通过受治理约束的 workflow 来创建，而不是靠无结构起草时，使用这个 capability。

## 触发条件

- 出现新的 capability 想法
- 这个 capability 需要显式阶段与门控
- 需要双语真源/派生文档
- 需要尽早规划 runtime-aware packaging

## 核心动作

1. 重述意图。
2. 定义边界。
3. 把工作切分为阶段。
4. 为每个阶段定义约束。
5. 定义跨阶段门控。
6. 判断是否可以进入下一阶段。
7. 先写英文真源，再派生中文。

## Guardrails

- 不经门控审查不得跳阶段。
- 不要把中文当成第二真源。
- 在 contract 不存在前，不要先宣称 installer 或 runtime 支持。
